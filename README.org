* trg - a pleasant Sierpinski gasket display
** Prerequisites
The GUI library, ltk, used by trg expects tk to be installed.
** Usage
Using [[https://www.quicklisp.org/beta/][Quicklisp]], clone trg into ~/quicklisp/local-projects/, then:

#+BEGIN_SRC
> (ql:quickload :trg)
... stuff happens
> (trg:main)
#+END_SRC
